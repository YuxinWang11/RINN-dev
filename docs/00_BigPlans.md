    - [x]  首先,将所有的涉及回推评估的画图,全部改成标准形式
    - [x]  实行彻底的Z改进方案:
        - 目前方案:为500组XY对生成500组Z,硬性拼接.
        - 不要给每个`(x,y)`固定一个`z`，而是**在每个训练 epoch，给所有`(x,y)`对重新采样一组全新的`z`**。
        - 原理：相当于每个 epoch，网络都在学习全新的`(x,y,z)`配对，不会死记硬背固定的`z`，被迫学到`x→y`的核心映射，以及`z`的分布规律。训练 100 个 epoch，网络相当于见过了 5 万组不同的`(x,y,z)`样本，完全足够建模`z`的分布。
